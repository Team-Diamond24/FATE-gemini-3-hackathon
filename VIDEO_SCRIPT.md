## Recording Instructions

### Before Recording

1. **Prepare Test Account**
   ```bash
   # Start the app
   npm run dev
   
   # Create account with email: demo@fate.com
   # Complete preferences setup
   # Play through 1-2 months to have history
   ```

2. **Browser Setup**
   - Use Chrome or Edge (best performance)
   - Set zoom to 100%
   - Hide bookmarks bar: Ctrl+Shift+B
   - Close all other tabs
   - Disable notifications
   - Use Incognito mode for clean UI

3. **Screen Recording Setup (OBS Studio)**
   - Resolution: 1920x1080
   - Frame Rate: 30fps
   - Bitrate: 2500-5000 kbps
   - Audio: 192kbps, 48kHz
   - Format: MP4 (H.264)

4. **Audio Setup**
   - Test microphone levels
   - Use pop filter if available
   - Record in quiet room
   - Speak clearly and at moderate pace
   - Practice narration beforehand

### During Recording

1. **Take Multiple Takes**
   - Record each segment separately
   - Do 2-3 takes of each segment
   - Choose best take in editing

2. **Pacing**
   - Speak at 150-160 words per minute
   - Pause briefly between sentences
   - Match narration to on-screen actions

3. **Mouse Movement**
   - Move cursor smoothly
   - Hover to highlight important elements
   - Don't move too fast

4. **Transitions**
   - Leave 1-2 seconds between segments
   - Use smooth page transitions
   - Avoid jarring cuts

### After Recording

1. **Video Editing (DaVinci Resolve / CapCut)**
   - Import all clips
   - Arrange in timeline
   - Cut out mistakes and pauses
   - Add transitions (fade, dissolve)
   - Add text overlays
   - Add background music (low volume)
   - Color correction if needed
   - Export as MP4 (H.264, 1080p, 30fps)

2. **Audio Editing**
   - Remove background noise
   - Normalize audio levels
   - Add subtle background music
   - Ensure narration is clear

3. **Text Overlays**
   - Use clean, readable font (Arial, Roboto)
   - White text with black outline
   - Position: Bottom third or top right
   - Duration: 2-3 seconds each
   - Fade in/out animations
